规模化运行 AI 智能体所需的一切

从个人开发者到企业集群 — 一个工具，零复杂度。

一键部署

交互式向导或 CLI 参数 — 自动生成 Docker Compose，容器启动，仪表盘就绪。60 秒内完成。

网络隔离

双网络架构：管理平面与工作节点隔离。工作节点无法访问控制层。

实时仪表盘

基于 SSE 的实时视图 — 健康状态、日志、资源使用和重启记录。CLI 或浏览器均可。

默认安全

时间安全的 API Key 认证，每个操作均有审计日志，实例间数据隔离。

零停机升级

先创建新容器再移除旧容器。版本锁定，安全回滚。无需手动操作 Docker。

按需扩展

3 个实例免费。团队版支持 50 个。企业版无上限。同一 CLI，同一仪表盘，同一工作流。
